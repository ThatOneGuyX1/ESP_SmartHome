Notes for getting the code to run on your devices

- I am using MicroPython v1.27. I've tested with v1.28 and that works as well.
- In my implementation, I've shortened smart_esp_comm.py to common_esp.py. Please change this when saving to your board.
- Upload the host code to one board and node code to another board. I named each file "main.py" when saving to respective boards.

- I have the host plugged into Thonny so I can see packets coming through. You should see a packet every 5 seconds.

- Please reach out to John Gahnz if you have any questions!
